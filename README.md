

# 🥗 Nutri App

Aplicação desktop desenvolvida em Python utilizando Tkinter.

O objetivo do app é calcular:

- IMC (Índice de Massa Corporal)
- TMB (Taxa Metabólica Basal)
- GET (Gasto Energético Total)

---

## 🚀 Funcionalidades

✅ Sistema de múltiplas páginas  
✅ Menu lateral moderno  
✅ Cálculo automático de IMC  
✅ Cálculo de TMB baseado na fórmula de Mifflin-St Jeor  
✅ Cálculo de GET com nível de atividade  
✅ Interface em modo escuro  

---

## 🛠 Tecnologias utilizadas

- Python 3
- Tkinter (GUI)
- ttk (Combobox)
